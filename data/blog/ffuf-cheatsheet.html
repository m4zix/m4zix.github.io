FFUF (Fuzz Faster U Fool) Cheatsheet

A professional guide to mastering FFUF for directory brute-forcing, VHost detection, and fuzzing parameters in ethical hacking assessments.

üö© Flag Reference

  • -u: Target URL (use FUZZ where input is injected)
  • -w: Wordlist to use for fuzzing
  • -X: HTTP method (e.g., GET, POST)
  • -d: Request data for POST requests
  • -H: Add HTTP headers
  • -mc: Match HTTP status codes
  • -fc: Filter HTTP status codes
  • -fs: Filter response size
  • -fw: Filter by number of words in response
  • -fr: Filter using regex on response content
  • -e: Add file extensions to test
  • -x: Forward traffic via proxy
  • -o: Output results to file
  • -of: Output file format (e.g., json, html)
  • -replay-proxy: Replay only matched results through proxy

üîß Basic Enumeration

Fuzz common directories using a large wordlist

ffuf -u http://target.local/FUZZ -w /usr/share/seclists/Discovery/Web-Content/big.txt

Fuzz with a custom placeholder (e.g., NORAJ)

ffuf -u http://target.local/NORAJ -w /usr/share/seclists/Discovery/Web-Content/big.txt:NORAJ

Find generic files and directories

ffuf -u http://target.local/FUZZ -w /usr/share/seclists/Discovery/Web-Content/raft-medium-files-lowercase.txt

Detect technologies using index extensions

ffuf -u http://target.local/indexFUZZ -w /usr/share/seclists/Discovery/Web-Content/web-extensions.txt

Fuzz words with identified extensions

ffuf -u http://target.local/FUZZ -w /usr/share/seclists/Discovery/Web-Content/raft-medium-words-lowercase.txt -e .php,.txt

Fuzz directories specifically

ffuf -u http://target.local/FUZZ -w /usr/share/seclists/Discovery/Web-Content/raft-medium-directories-lowercase.txt

üìÑ Filters

Hide HTTP 403 Forbidden responses

ffuf -u http://target.local/FUZZ -w raft-medium-files-lowercase.txt -fc 403

Show only successful (200 OK) responses

ffuf -u http://target.local/FUZZ -w raft-medium-files-lowercase.txt -mc 200

Match 500 Internal Server Error responses

ffuf -u http://target.local/FUZZ -w raft-medium-files-lowercase.txt -mc 500

Filter zero-size responses (often empty)

ffuf -u http://target.local/config/FUZZ -w raft-medium-files-lowercase.txt -fc 403 -fs 0

Use regex to exclude results (e.g., dot files)

ffuf -u http://target.local/FUZZ -w raft-medium-files-lowercase.txt -fr '/\..*'

üîé Parameter Fuzzing

Find injectable parameter names using wordlist

ffuf -u 'http://target.local/page.php?FUZZ=1' -c -w burp-parameter-names.txt -fw 39

Fuzz numeric values in parameters

seq 0 255 | ffuf -u 'http://target.local/page.php?id=FUZZ' -c -w - -fw 33

POST brute-force login passwords

ffuf -u http://target.local/login -c -w passwords.txt -X POST -d 'username=admin&password=FUZZ' -fs 1435 -H 'Content-Type: application/x-www-form-urlencoded'

üåê Subdomain & VHost Discovery

Brute-force subdomains

ffuf -u http://FUZZ.target.local -c -w subdomains.txt

VHost discovery via Host header injection

ffuf -u http://target.local -c -w vhosts.txt -H 'Host: FUZZ.target.local' -fs 0

üß≠ Proxying

Forward traffic through proxy (e.g., Burp)

ffuf -u http://target.local/FUZZ -c -w common.txt -x http://127.0.0.1:8080

Replay only matched responses through proxy

ffuf -u http://target.local/FUZZ -c -w common.txt -replay-proxy http://127.0.0.1:8080

1. Hidden GET Parameters

ffuf -u 'http://target.local/page.php?FUZZ=1' -w burp-parameter-names.txt -fw 50

ffuf -u http://target.local/admin/FUZZ -w common.txt -e .php,.bak,.old,.zip,.inc,.swp,.backup

3. Auth Brute Force (POST)

ffuf -u http://target.local/login -X POST -d 'user=admin&pass=FUZZ' -H 'Content-Type: application/x-www-form-urlencoded' -w rockyou.txt -fc 403

4. HTTP Method Fuzzing

ffuf -X FUZZ -u http://target.local/ -w http-methods.txt -mc 200

5. API Route Discovery

ffuf -u http://target.local/api/FUZZ -w api-endpoints.txt -H 'Authorization: Bearer <token>' -mc 200

6. Common Parameter Value Fuzzing

ffuf -u 'http://target.local/page.php?debug=FUZZ' -w booleans.txt -mc 200

7. JSON API Login Bruteforce

ffuf -u http://target.local/api/login -X POST -w passwords.txt -H 'Content-Type: application/json' -d '{"username": "admin", "password": "FUZZ"}' -fs 200

8. Internal Directory Enumeration

ffuf -u http://target.local/FUZZ/ -w directories.txt -fc 403,404 -fs 0

9. Open Redirect Fuzzing

ffuf -u 'http://target.local/redirect.php?url=FUZZ' -w open-redirect-payloads.txt -mc 302

10. Command Injection Parameter Check

ffuf -u 'http://target.local/cmd.php?FUZZ=ls' -w command-parameters.txt -fw 22

ffuf -u http://target.local -H "Host: FUZZ.target.local" -w vhosts.txt -fs 0

ffuf -u http://target.local/upload.php -X POST -w extensions.txt -d 'filename=shell.FUZZ' -mc 200

14. Header Injection Fuzzing

ffuf -u http://target.local/protected -w headers.txt -H 'FUZZ: admin' -mc 200

15. Export JSON for Scripts
